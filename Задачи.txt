 - Target: По нажатию на элемент ему задается класс который служит целью для применения стилей
 - Side Bar: Боковая панель с сылками на каждый из созданых элементов по нажатию на который ему задается таргет для быстрого доступа
 - Scale: При наведении на блок реализовать разтягивание блока
 - Stile Bar: Панель с стиляси которые применяются для цели Target вроде цвета, шрифта, ориентации Flex, заливки, закругления
 - Frame Size: В меню создать настройки ширины холста с предустановлеными значениями
 - Corner Radius: Создать механику закругления углов
 - Drop Down List Menu: Создать меню с выпадающим списком по категориям
 - Delete: Реализовать удаление элемента target по нажатию delete
 - Mouse Menu: Меню по нажатию правой кнопкой мыши для элемента target
 - History: Реализовать историю всех действий и востановление изменений
 - Preview: Реализовать функцию которая скопирует все стили и элементы из окна и откроет превью версию в новом окне без меню (Чисто код из области редактора)
